# Change Log

All notable changes to the "SwitchBase" extension will be documented in this file.

## [1.0.0] - 2025-11-13

### Added
- Smart number format detection and preservation
- Support for programming language suffixes (u, UL, f, d, etc.)
- Grouped submenu for cleaner context menu experience
- Silent operation mode (no success notifications)
- Enhanced error handling and validation
- Automatic base detection for hex and binary numbers

### Changed
- Improved user interface with organized menu structure
- Better number parsing with format preservation
- Updated extension metadata for marketplace

### Fixed
- Memory issues with large codebases
- Proper handling of various number formats
